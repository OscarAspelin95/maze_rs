# maze_rs
Desktop application for visualizing maze solver algorithms.

Built using the [Dioxus](https://github.com/DioxusLabs/dioxus) framework.


## Usage
The easiest way to get started is to download the `.AppImage` application from the [releases page](https://github.com/OscarAspelin95/maze_rs/releases).

## Development

### Requirements
- Linux OS (Ubuntu 24.04.2)
- Rust 2021 edition
- Dioxus cli == 0.7.2

### Installation
Make sure `cargo` and `dioxus-cli` are installed.

### Run
`make run` - build and spin up a local desktop application
`make bundle-linux` - build and create a Linux bundle.
